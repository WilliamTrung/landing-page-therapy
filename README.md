# Therapy Studio Landing Page

Static HTML website for Therapy Studio, a Japanese landing page and blog prototype for a women-focused therapy and salon training service.

## Pages

- `landing_page.html` - Main landing page with course sections, blog highlights, schedule, amenity details, and reservation form.
- `blog_list.html` - Blog listing page with article cards and pagination behavior.
- `blog_detail.html` - Blog detail page with article content and edit link.
- `blog_create.html` - Blog creation/admin-style page with editor mockup and article previews.

## Usage

Open `landing_page.html` in a browser to view the site locally. All pages are plain HTML/CSS/JavaScript and do not require a build step.
